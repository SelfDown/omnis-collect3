result_handler 结果处理
=========================================
result_handler 对模块处理后的结果进行处理


    * 比如将只有一个对象的 **数组** ，**转** 成一个 **对象**
    * 将一个 **列表转成树** 形结构
    * 将 **2个服务** 的数据，**拼接** 在一起
    * 记录中添加一些字段
    * 记录中修改一些字段数据
    * 将 **结果** 进行 **排序**
    * 如果是树形结构数据，可以提取，一级、二级节点名称。比如底层部门显示公司一级、二级部门
    * 将结果添加到请求参数
    * 将参数转字段
    * 字段转json
    * 结果 **导出excel**
